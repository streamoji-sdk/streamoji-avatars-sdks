# Streamoji - React Avatar Creator

**Streamoji - React Avatar Creator** is a set of components and helper methods to implement the Streamoji Avatar Creator into React projects.

## Installation

Streamoji React Avatar Creator is available as an npm package.

```bash
npm i @streamoji/react-avatar-creator
```

## Usage

```tsx
import { AvatarCreator } from '@streamoji/react-avatar-creator';

export default function App() {
  return <AvatarCreator token="YOUR_BACKEND_AUTH_TOKEN" style={{ width: '100%', height: '100vh', border: 'none' }} />;
}
```

# Components

## AvatarCreator

`AvatarCreator` component helps you load the Streamoji Avatars editor in an iframe where users can customize their avatar securely and you receive their exported avatar URL upon completion.

### Parameters

**token** _[required]_: string
- The auth token generated by your backend using Streamoji credentials.

**className** _[optional]_: string
- The css classes to apply to the iframe.

**style** _[optional]_: CSSProperties
- The css styles to apply to the iframe.

**config** _[optional]_: AvatarCreatorConfig
- Editor Configuration where you can set URL properties of the Streamoji editor. Available settings:
  - `bodyType`: `'Half' | 'Full'` restricts the builder to a specific body type.
  - `saveConfirm`: boolean shows a confirmation modal before finalizing the export.
  - `whiteLabelTitle`: string customizes the title displayed inside the editor.
  - `whiteLabelColor`: string applies a custom theme color (Hex format, e.g. 'BB86FC' or '#BB86FC').
  - `thumbnail`: boolean if set to true, includes `thumbnailUrl` in the export event payload.

**onReady** _[optional]_: () => void
- Callback function triggered after the iframe announces `v1.frame.ready` and the SDK subscribes to events.

**onAvatarExported** _[optional]_: (data: AvatarExportedData) => void
- Callback function triggered when avatar export completes.

### Example

```tsx
import { useEffect, useState } from 'react';
import { AvatarCreator, type AvatarExportedData } from '@streamoji/react-avatar-creator';

export default function App() {
  const [token, setToken] = useState('');

  useEffect(() => {
    fetch('/api/get-avatar-token')
      .then((response) => response.json())
      .then((data) => setToken(data.token));
  }, []);

  const handleOnAvatarExported = (data: AvatarExportedData) => {
    console.log(`Avatar successfully compiled! URL: ${data.url}`);
    if (data.thumbnailUrl) {
      console.log(`Thumbnail preview: ${data.thumbnailUrl}`);
    }
  };

  if (!token) return <div>Loading token...</div>;

  return (
    <AvatarCreator
      token={token}
      config={{
        bodyType: 'Full',
        thumbnail: true,
        saveConfirm: true,
        whiteLabelTitle: 'My Custom App',
        whiteLabelColor: 'BB86FC',
      }}
      style={{ width: '100%', height: '100vh', border: 'none' }}
      onReady={() => console.log('Creator ready')}
      onAvatarExported={handleOnAvatarExported}
    />
  );
}
```

## Authentication

Generate the Streamoji auth token on your backend, then pass the returned token into `AvatarCreator`.

```js
const getStreamojiToken = async (userId, userName) => {
  const response = await fetch('https://us-central1-streamoji-265f4.cloudfunctions.net/getAuthToken', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Client-Secret': process.env.STREAMOJI_CLIENT_SECRET,
      'Client-Id': process.env.STREAMOJI_CLIENT_ID,
    },
    body: JSON.stringify({ userId, userName }),
  });

  const data = await response.json();
  return data.authToken;
};
```

## AvatarCreatorRaw

`AvatarCreatorRaw` is a lower-level component that establishes the iframe and message-bus connections without abstracting away explicit event callbacks. You receive all broadcast messages emitted by Streamoji via a single hook. 

### Parameters

All parameters from `AvatarCreator` exist here, replacing explicit interaction callbacks with `onEventReceived`:

**token** _[required]_: string
- The auth token generated by your backend.

**onReady** _[optional]_: () => void
- Callback function triggered after the iframe announces `v1.frame.ready`.

**onEventReceived** _[optional]_: (event: IFrameEvent<any>) => void
- Standard callback that intercepts any `v1.**` namespace events published inside the Avatar Creator.

### Example

```tsx
import { AvatarCreatorConfig, IFrameEvent, AvatarCreatorRaw } from '@streamoji/react-avatar-creator';

const config: AvatarCreatorConfig = {
  bodyType: 'Half',
  saveConfirm: true,
  whiteLabelTitle: 'Streamoji SDK',
  whiteLabelColor: '#3B82F6'
};

const style = { width: '100%', height: '100vh', border: 'none' };

export default function App() {
  const handleCustomEvent = (event: IFrameEvent) => {
    console.log(`Received raw Custom Event structure:`, event);
  };

  return (
    <>
      <AvatarCreatorRaw 
         token="YOUR_BACKEND_AUTH_TOKEN"
         config={config} 
         style={style} 
         onEventReceived={handleCustomEvent} 
      />
    </>
  );
}
```
